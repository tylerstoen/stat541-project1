Data Context

plastics <- readr::read_csv('https://raw.githubusercontent.com/rfordatascience/tidytuesday/main/data/2021/2021-01-26/plastics.csv')

This dataset comes from global plastic waste cleanup efforts organized by Break Free From Plastic, where volunteers collect and categorize plastic items found in the environment through “brand audits.” The cleaned version of the data was compiled and published by Sarah Sauvé, who documented her approach in a blog post and shared the dataset to raise awareness and support broader efforts to understand plastic pollution sources. The data were collected across multiple countries worldwide, and each row represents the outcome of a cleanup for a specific country, year (2019 or 2020), and parent company, which serves as the identified source of the plastic. The dataset records counts of different types of plastic materials, including HDPE (high-density polyethylene), LDPE (low-density polyethylene), PET (polyester), polypropylene, polystyrene, PVC, and other, with a grand total variable representing the sum of all plastic items collected. These data were gathered during organized cleanup events in which volunteers manually collected waste and categorized items by material type and branding, allowing for company-level attribution when possible.

With this dataset, we are able to look and study how plastic waste varies by geography, material type and source company across a two year period and to better understand the sources and composition of plastic pollution globally. By identifying which companies contribute most to plastic waste and how waste varies across countries and materials, the dataset supports efforts to raise awareness, inform policy, and encourage corporate accountability. The data are of particular interest because plastic pollution is a major environmental issue, and understanding its sources is a critical step toward reducing its impact.

The data was cleaned by first combining multiple country-level CSV files for each year into a single dataset using row-binding. Column types were explicitly specified to ensure consistency across files, with categorical variables stored as character and count variables as numeric. A year variable was added to distinguish observations from 2019 and 2020. The grand total variable was converted from character to numeric using a parsing function, parse_number(), to remove punctuation. Names were also converted to lowercase. The 2019 data required additional cleaning to resolve inconsistencies, including extracting country names from file names, renaming variables to match the 2020 structure, and combining duplicated columns (e.g., pp and pp_2, ps and ps_2). After merging duplicate columns, the redundant variables were removed. These steps resulted in a unified dataset with consistent structure across both years.

On top of the data cleaning done as preparation, there were some observations with missing data. In many of the cases, plastic was all classified under “other”, which provides an incorrect split of plastic types. This proves problematic for visualizing things like the split of plastic types because these plastics are almost certainly classified incorrectly. It would be misleading to replace these with zeros when investigating different plastic types, so instead we likely have to drop these rows to maintain correctness.

Data from API

API 1: restcountries.com

The additional country-level data were retrieved from the REST Countries API, which is a free, community-maintained RESTful API that aggregates geographic and demographic information for over 250 countries from various public sources. The data represent country-level attributes, so the observational unit is a country, with variables such as region, subregion, United Nations membership status, population density, and geographic coordinates. Unlike the plastics dataset, these data are not tied to a specific year; instead, they reflect the current state of country metadata at the time of access, as the API is continuously updated. The data were collected programmatically by sending an HTTP request to the API endpoint using the httr package in R, and the returned JSON data were parsed and converted into a tidy format using jsonlite. We chose to include this dataset to provide additional geographic and geo-political context for the countries in the plastic pollution data, allowing us to group, compare, and visualize patterns in plastic waste collection across regions and levels of development.

## Fuzzy matching for parent companies using stringdist

noise_words <- c(
  "\\bthe\\b",
  "\\binc\\.?\\b",
  "\\bco\\.?\\b",
  "\\bltd\\.?\\b",
  "\\bcorp\\.?\\b",
  "\\bcompany\\b",
  "\\bcompanies\\b",
  "\\bgroup\\b",       # "Nestlé Group" -> "nestlé"
  "\\bholdings\\.?\\b",
  "\\bllc\\.?\\b",
  "\\bplc\\.?\\b",
  "\\bgmbh\\b",        # common German suffix
  "\\bsa\\b",          # common in Latin America/Europe
  "\\bsrl\\b"
)

normalize_company <- function(x) {
  x %>%
    str_to_lower() %>%
    str_remove_all(paste(noise_words, collapse = "|")) %>%
    str_remove_all("[^a-z0-9 ]") %>%
    str_squish()
}

resolve_typos <- function(names, threshold = 0.1) {
  dist_matrix <- stringdistmatrix(names, names, method = "jw")
  diag(dist_matrix) <- Inf
  
  canonical <- names
  
  for (i in seq_along(names)) {
    best_dist <- min(dist_matrix[i, ])
    if (best_dist < threshold) {
      best_match <- which.min(dist_matrix[i, ])
      canonical[i] <- min(names[i], names[best_match])
    }
  }
  
  tibble(normalized = names, canonical = str_to_title(canonical))
}

companies_raw <- unique(joined$parent_company)

name_map <- tibble(original = companies_raw) %>%
  mutate(normalized = normalize_company(original)) %>%
  left_join(
    resolve_typos(unique(.$normalized)),  # fuzzy match on normalized names only
    by = "normalized"
  )

canonical_labels <- name_map %>%
  count(canonical, original) %>%          # count how often each original appears

API 2: worldbank

We retrieved total population data and GDP data (in current U.S. dollars) for the years corresponding to collection years in the plastics dataset (2019 and 2020) from the World Bank’s World Development Indicators (WDI), a comprehensive collection of country-level development statistics compiled from official international sources. The data were published by the World Bank, with GDP and population estimates produced by World Bank country economists using information from national statistical agencies, the OECD, and other official country sources. The observational unit for this dataset is a country–year combination, matching the structure of the plastics data. The data correspond specifically to the years 2019 and 2020, aligning with the time frame of the cleanup records. These data were collected programmatically by sending requests to the World Bank Indicators REST API, and the returned JSON data were parsed and converted into a tidy format. We chose to include this dataset because it provides important economic context for the countries where plastic waste was collected, allowing us to explore how factors such as GDP and population relate to patterns in plastic waste across countries.

calc_prop <- function(var, data) {
  data %>%
    filter(grand_total > 0, parent_company != "Grand Total") %>%
    group_by(year) %>%

Figure 1 displays each plastic type as a proportion of total plastic collected, comparing 2019 and 2020. The most striking pattern is the dominance of the “Other” category, which accounts for approximately 60% of all recorded plastic in 2019. Notably, this percentage dips to below 40% in 2020. A shift of this magnitude in a year time frame is unlikely to reflect a true change in plastic distribution, and more plausibly reflects improved plastic type recording between years. Among the named types, PET and PP represent the largest shares, which is consistent with their widespread use in beverage bottles and food packaging. Other plastic types such as PVC and Polystyrene make up a very small percentage of collected waste, possibly due to their niche use and typically larger size. Overall, these results indicate that plastic waste is not evenly distributed across types, but instead concentrated among a few categories, highlighting important patterns in material waste distribution.

This dataset comes from global plastic waste cleanup efforts organized by [Break Free From Plastic]{.underline}, where volunteers collect and categorize plastic items found in the environment through “brand audits.” The cleaned version of the data was compiled and published by [Sarah Sauvé]{.underline}, who documented her approach in a blog post and shared the dataset to raise awareness and support broader efforts to understand plastic pollution sources. The data were collected across multiple countries worldwide, and each row represents the outcome of a cleanup for a specific country, year (2019 or 2020), and parent company, which serves as the identified source of the plastic. The dataset records counts of different types of plastic materials, including HDPE (high-density polyethylene), LDPE (low-density polyethylene), PET (polyester), polypropylene, polystyrene, PVC, and other, with a grand total variable representing the sum of all plastic items collected. These data were gathered during organized cleanup events in which volunteers manually collected waste and categorized items by material type and branding, allowing for company-level attribution when possible.

With this dataset, we are able to look and study how plastic waste varies by geography, material type and source company across a two year period and to better understand the sources and composition of plastic pollution globally. By identifying which companies contribute most to plastic waste and how waste varies across countries and materials, the dataset supports efforts to raise awareness, inform policy, and encourage corporate accountability. The data are of particular interest because plastic pollution is a major environmental issue, and understanding its sources is a critical step toward reducing its impact.

The data was cleaned by first combining multiple country-level CSV files for each year into a single dataset using row-binding. Column types were explicitly specified to ensure consistency across files, with categorical variables stored as character and count variables as numeric. A year variable was added to distinguish observations from 2019 and 2020. The grand total variable was converted from character to numeric using a parsing function, parse_number(), to remove punctuation. Names were also converted to lowercase. The 2019 data required additional cleaning to resolve inconsistencies, including extracting country names from file names, renaming variables to match the 2020 structure, and combining duplicated columns (e.g., pp and pp_2, ps and ps_2). After merging duplicate columns, the redundant variables were removed. These steps resulted in a unified dataset with consistent structure across both years.

On top of the data cleaning done as preparation, there were some observations with missing data. In many of the cases, plastic was all classified under "other", which provides an incorrect split of plastic types. This proves problematic for visualizing things like the split of plastic types because these plastics are almost certainly classified incorrectly. It would be misleading to replace these with zeros when investigating different plastic types, so instead we likely have to drop these rows to maintain correctness.

resolve_typos <- function(names, threshold = 0.1) {
  dist_matrix <- stringdistmatrix(names, names, method = "jw")
  diag(dist_matrix) <- Inf
  
  canonical <- names
  
  for (i in seq_along(names)) {
    best_dist <- min(dist_matrix[i, ])
    if (best_dist < threshold) {
      best_match <- which.min(dist_matrix[i, ])
      canonical[i] <- min(names[i], names[best_match])
    }
  }
  
  tibble(normalized = names, canonical = str_to_title(canonical))
}
